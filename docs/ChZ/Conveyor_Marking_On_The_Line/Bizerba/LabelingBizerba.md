# Работа с маркировкой на оборудовании Bizerba

Выгрузка заданий по маркировочным линиям осуществляется в АРМе «Маркировка bizerba». Строки попадают в этот АРМ на основании производственных заданий. Для начала работы потребуется создать "Производственное задание" и на его основании "Задание на маркировку". 

Для того, что открыть АРМ, требуется перейти в подсистему **"Интеграция с Bizerba Brain2"** и выбрать **"Маркировка Bizerba"**.

![](LabelingBizerba.assets\image.png)

Отбор заданий происходит по дате смены (Дата смены должна быть в рамках периода) и по Рабочему центру (Рабочий центр из задания должен равняться Рабочему из отбора в АРМе).

![](LabelingBizerba.assets\image2.png)

## Выгрузка заданий

Для выгрузки заданий на линию нужно выбрать номер линии для выгрузки.

![](LabelingBizerba.assets\image3.png)

Затем необходимо выбрать нужную строку и нажать кнопку «Выгрузить задание».

![](LabelingBizerba.assets\image4.png)

Если выгрузка прошла успешно, то появится уведомление и проставится галка в столбце «Выгружено».

## Маркировка продукции

После маркировки продукции на линии в АРМе отразится информация о количестве нанесенных кодов в колонке «Промаркировано». Если «План» равен «Промаркировано», то ставится галочка в столбце «Маркировка завершена».

![](LabelingBizerba.assets\image5.png)

Если в столбце «Маркировка завершена» стоит галка, то можно нажимать «Подтвердить задание».

![](LabelingBizerba.assets\image6.png)

При этом будут созданы документы:
- «Переработка» - отражает производственный передел;
- «Маркировка товаров ИС МП» - отражает факт расхода кодов.
