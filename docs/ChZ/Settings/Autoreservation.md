# Автоматическое резервирование кодов под Задание на маркировку

У заданий на маркировку существует несколько статусов: *К обеспечению, Не требует обеспечения, Обеспечено*.

Для автоматического резервирования кодов под задание предназначено регламентное задание **"ИС МП. Авторезервирование кодов под задания на маркировку"**.

[![1][1]][1]

Под действие регламентного задания попадают все Задания на маркировку, имеющие статус "К обеспечению". В случае наличия свободных кодов в пуле, необходимое количество будет зарезервировано под задание, а статус задания изменится на "Обеспечено". Если свободных к резерву кодов нет, то задание не претерпит никаких изменений.

Статус "Не требует обеспечения" необходим для заданий ручного резерва и контроля оператора.
